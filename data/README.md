## Datasets

### Source

- **Online Retail Dataset I:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/352/online+retail)
- **Online Retail Dataset II:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/502/online+retail+ii)

> **Note:** Both datasets are publicly available. To keep the repository size manageable, **do not commit large data files**. For local work, download/extract the datasets and/or generate processed datasets in the appropriate folder and keep them untracked by Git (via `.gitignore`).
