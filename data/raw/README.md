## Datasets

### Source

- **Online Retail Dataset I:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/352/online+retail)
- **Online Retail Dataset II:** [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/502/online+retail+ii)

> **Note:** Both datasets are publicly available. To keep the repository size manageable, **do not commit raw data files**. For local work, download and extract the datasets into this folder and ensure they are untracked by Git (e.g., add them to `.gitignore`).
